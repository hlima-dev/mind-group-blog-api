# MindGroup Blog

Uma plataforma moderna de publicação de artigos desenvolvida com React, TypeScript e Node.js.

## 🚀 Tecnologias

### Frontend
- React
- TypeScript
- Vite
- TailwindCSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express
- TypeScript
- MySQL
- JWT Authentication
- Multer

## ✨ Funcionalidades

- Autenticação JWT
- Cadastro e login de usuários
- Publicação de artigos
- Upload de banner
- Dashboard administrativa
- Sistema de categorias
- Busca de artigos
- Layout responsivo
- Interface moderna
